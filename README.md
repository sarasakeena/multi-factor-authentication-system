# AI-Based Multi-Factor Authentication System

## Overview
This project is a multi-layered authentication system designed to improve security using AI/ML techniques. It combines biometric verification, device fingerprinting, and behavioral analysis to provide a secure and seamless login experience.

## Key Features
- 🔐 Multi-factor authentication (biometrics + device + behavior)
- 👤 Face recognition using AI models
- 📱 Device fingerprinting to detect unknown devices
- 🧠 AI-based anomaly detection (login patterns, behavior)
- 🚫 Protection against phishing and credential attacks
- ⚡ Real-time authentication decisions

## Tech Stack
- Python, Flask
- OpenCV, DeepFace, face_recognition
- TensorFlow / Keras
- PostgreSQL
- Docker

## System Architecture
- Frontend: User login & biometric capture
- Backend: Authentication logic and API
- AI/ML Engine: Face recognition & anomaly detection
- Database: Stores user data, embeddings, logs
- Security Layer: Encryption using SHA-512

## How It Works
1. User attempts login
2. Face recognition verifies identity
3. Device fingerprint is checked
4. Behavior patterns are analyzed
5. System decides authentication result
