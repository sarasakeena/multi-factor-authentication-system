# AI-Based Multi-Factor Authentication System

## Overview
This project is a multi-factor authentication system that enhances security using facial recognition and additional verification techniques. It was developed during a hackathon.

## Features
- Face recognition using OpenCV
- Liveness / image validation
- Multi-step authentication flow
- Device fingerprinting
- Chatbot-based interaction (if applicable)

## Tech Stack
- Python
- OpenCV
- Flask
- PostgreSQL
- Docker

## Project Structure
- `src/` - Core logic
- `templates/` - Frontend UI
- `static/` - Assets
- `models/` - ML models
- `database/` - DB related files

## How It Works
1. User provides input (image/video)
2. Face is detected and verified
3. Additional checks (liveness/device validation)
4. Authentication decision is returned
